> Ensure adoption of flowtype

## Usage

Install:

```sh
yarn add danger-plugin-flow --dev
```

At a glance:

```js
// dangerfile.js
import { schedule } from "danger";
import flow from "danger-plugin-flow";

schedule(flow());
```
